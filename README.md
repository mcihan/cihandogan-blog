# Cihan Dogan - Personal Portfolio & Blog

Static personal website and technical blog. Built with pure HTML, CSS, and minimal JavaScript. Hosted on GitHub Pages at [cihandogan.co.uk](https://cihandogan.co.uk).

## Tech Stack

- **HTML5** (semantic markup)
- **CSS3** (custom, no frameworks)
- **Vanilla JS** (mobile menu toggle only)
- **GitHub Pages** (hosting)
- System fonts (no external font loading)

## Pages

| Page | File | Description |
|------|------|-------------|
| Home | `index.html` | Landing page with hero, expertise grid, stats, and featured article |
| About | `about.html` | Professional biography, career timeline, skills, and education |
| Articles | `blog.html` | Blog listing page with article previews |
| Article | `articles/*.html` | Individual article pages |
| Contact | `contact.html` | Contact information and social links |

## Setup

1. Add profile photo to `assets/images/profile.jpg`
2. Add article images to `assets/images/`
3. Push to GitHub and enable GitHub Pages
4. Set custom domain to `cihandogan.co.uk`
5. Configure DNS A record at GoDaddy
